

# 布局设置

## 默认布局

`NexT`提供了`4`种布局(或称为主题、`scheme`)

    Muse 主题
    Mist 主题
    Pisces 主题
    Gemini 主题

预览界面：[即时预览](https://github.com/theme-next/hexo-theme-next/tree/master/docs/zh-CN#%E5%8D%B3%E6%97%B6%E9%A2%84%E8%A7%88)

## 设置

在`NexT _config.yml`文件中修改

    # Schemes
    # scheme: Muse
    # scheme: Mist
    # scheme: Pisces
    scheme: Gemini

*当前使用`Gemini`*
