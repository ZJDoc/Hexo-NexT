
# 加载进度条

网页加载阶段的进度显示，提高用户操作体验

## 配置

`NexT _config.yml`默认已开启了`nprogress`

```
# Progress bar in the top during page loading.
# For more information: https://github.com/rstacruz/nprogress
nprogress:
  enable: true
  spinner: true
```
