
# 代码块设置

## 代码高亮

`Hexo`默认情况下启用`highlight.js`，在`Hexo`中用作服务器端高亮显示

```
highlight:
  enable: true
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: true
  hljs: false
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''
```
